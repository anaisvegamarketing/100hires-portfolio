# 100Hires Portfolio Project

## Tools Installed

- **Cursor IDE** — AI-powered code editor (https://cursor.com/)
- **Claude Code for VS Code** — AI coding assistant by Anthropic
- **Codex (OpenAI)** — AI coding agent extension
- **Git 2.53.0** — Version control system (https://git-scm.com/)

## Steps Completed

1. Created a public GitHub repository named `100hires-portfolio`
2. Installed Cursor IDE on Windows
3. Installed Claude Code and Codex extensions from the Cursor marketplace
4. Installed Git for Windows to enable version control
5. Cloned the GitHub repository into Cursor using the "Clone repo" feature
6. Edited this README.md file directly in Cursor

## Issues Encountered and How I Solved Them

- **Git not recognized:** When I first opened the terminal in Cursor and ran `git --version`, I got a command not found error. I solved this by downloading and installing Git from https://git-scm.com/download/win, then restarting Cursor. After that, Git was recognized correctly.
